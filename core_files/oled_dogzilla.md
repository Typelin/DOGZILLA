# DOGZILLA OLED 顯示程式技術手冊 (oled_dogzilla.py)

本程式負責驅動機器狗機身上的 SSD1306 OLED 螢幕，提供系統狀態的即時視覺反饋。

## 一、 程式概述
* **檔案名稱**：`oled_dogzilla.py`
* **軟體版本**：V1.0.8
* **硬體規格**：128x32 像素單色 OLED (I2C 介面)

## 二、 依賴庫與模組引用
1.  **硬體驅動**：`Adafruit_SSD1306`（I2C 顯示器核心驅動）。
2.  **影像處理**：`PIL` (Pillow)（負責生成要顯示的位圖圖像、字型與繪圖）。
3.  **系統工具**：`subprocess`, `os`（執行 Linux 指令以獲取系統硬體狀態）。
4.  **機器人庫**：`DOGZILLALib`（用於讀取機器狗的電池電量）。

## 三、 功能章節
### 3.1 系統狀態抓取
程式透過 Linux Shell 指令獲取以下數據：
* **CPU Load**：讀取 `/proc/stat` 計算 CPU 使用率百分比。
* **RAM 狀態**：透過 `free` 指令解析記憶體使用率與剩餘容量。
* **Disk 狀態**：透過 `df` 指令解析 SD 卡的空間佔用情況。
* **IP 地址**：獲取目前 `wlan0` (WiFi) 或 `eth0` (有線) 的連線 IP。

### 3.2 影像繪製與更新
* **畫布結構**：建立一個 128x32 的黑白畫布。
* **多行顯示**：將畫布分為四行文字顯示空間。
* **電池圖示/文字**：(程式碼中預留功能) 可讀取並顯示電池百分比。

### 3.3 穩定性設計
* **熱插拔處理**：在 `main_program` 迴圈中包含 `try-except` 結構，防止 OLED 鬆脫導致程式崩潰。
* **更新頻率**：預設每 0.1 秒刷新一次內部數據，每 1 秒在無連線狀態下嘗試重連。

## 四、 顯示版面配置 (Layout)
* **第一行**：CPU 使用率 | 系統時間。
* **第二行**：RAM 使用百分比 -> 總容量 (GB)。
* **第三行**：SD 卡佔用率 -> 總容量 (GB)。
* **第四行**：`IPA:` 當前機器狗 IP 位址。
