# DOGZILLA 預設動作編號清單

本清單根據官方文檔 **「04.Performing actions.pdf」** 整理 。請使用 `bot.action(ID)` 函式呼叫。

### 1. 動作 ID 對照表

動作 ID 的有效輸入範圍為 **1 到 255** 。

| ID | 動作名稱 (中/英) | ID | 動作名稱 (中/英) |
| --- | --- | --- | --- |
| **1** | 趴下 (Lie Down) | **11** | 撒尿 (Pee) |
| **2** | 站起 (Stand Up) | **12** | 坐下 (Sit Down) |
| **3** | 匍匐前進 (Crawl) | **13** | 招手 (Wave Hand) |
| **4** | 轉圈 (Turn Around) | **14** | 伸懶腰 (Stretch) |
| **5** | 原地踏步 (Mark Time) | **15** | 波浪 (Wave Body) |
| **6** | 蹲起 (Squat) | **16** | 左右搖擺 (Swing) |
| **7** | 轉動 Roll | **17** | 求食 (Pray) |
| **8** | 轉動 Pitch | **18** | 找食物 (Seek) |
| **9** | 轉動 Yaw | **19** | 握手 (Handshake) |
| **10** | 三軸聯動 (3 Axis) | **255** | **恢復初始姿態 (Reset)** |

### 2. 測試程式碼範例 (test_action_final.py)

```python
import time
import sys

# 官方庫路徑導入
sys.path.append('/home/pi/DOGZILLA/app_dogzilla/')
from DOGZILLALib import DOGZILLA

def main():
    bot = DOGZILLA()
    bot.motor_speed(50) # 設定動作速度
    
    try:
        # 執行動作 1
        print("執行：招手 (ID: 13)")
        bot.action(13)
        time.sleep(4) # 必須等待，確保物理動作完成

        # 執行動作 2
        print("執行：握手 (ID: 19)")
        bot.action(19)
        time.sleep(4)

    except Exception as e:
        print(f"錯誤: {e}")
    finally:
        # 務必使用 255 作為結束重置，避免 Illegal Action ID 報錯
        print("恢復初始姿態 (ID: 255)")
        bot.action(255)
        time.sleep(2)
        del bot

if __name__ == "__main__":
    main()

```

---

### 3. 注意事項

* 
**安全重置**：程式結束前務必調用 `bot.action(255)`，這會停止所有操作並恢復初始姿勢 。


* **延遲補償**：由於硬體運轉需要時間，每個控制指令後建議加上 `time.sleep(1)` 以上的延遲。
* 
**動作輪播**：若要依序執行所有動作，可使用 `bot.perform(1)` 啟動輪播模式 。
